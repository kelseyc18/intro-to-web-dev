HTML content

HTML is a markup language that defines the structure of a web page.

<p>Some text here.</p>
<a href="https://www.google.com">Link to Google</a>

HTML content

                <div>
                  <p>Some text here.</p>
                  <p>Some <b>bold</b> text below.</p>
                </div>

HTML can be nested

                <!DOCTYPE html>
                <html>
                  <head>
                    <meta charset="utf-8"/>
                    <meta name="viewport" content="width=device-width">
                    <title>Add your title here!</title>
                  </head>
                  <body>
                    <p>Here is some <b>bold</b> text.</p>
                  </body>
                </html>

CSS styling

  • CSS is a language that defines the styles on a web page.
  • It can be used to specify the colors, fonts, and layouts for a web page.

A CSS rule

        h1 {
          font-size: 32px;
          color: blue;
        }

h1 is the selector

font-size and color are properties

32px and blue are values

CSS selectors

        <p>Reviews of Harry Potter and the Sorcerer's Stone</p>
        <p>Kelsey's review...</p>
        <p>Anna's review...</p>
        <p>Emily's review...</p>

CSS selectors

        <p>Reviews of Harry Potter and the Sorcerer's Stone</p>

        <p>Anna's review...</p>
        <p>Emily's review...</p>

Tag

CSS selectors

        <p class="review">Anna's review...</p>
        <p class="review">Emily's review...</p>

Class

CSS selectors

        <p class="review">...</p>
